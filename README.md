# Projeto de site Crowdfunding

### Nome do projeto: 
**Elísio**

### Banco de dados: 
A aplicação roda um banco de dados que possui Doadores, Doações e Projetos, de modo que doadores possam ajudar em projetos sociais através de doações. 

### Site: 
O site consiste numa página inicial de exposição dos projetos de doação alimentados no banco de dados, de modo que se pode procurar por palavras chave para achar os projetos, assim como também foi implementado sistema de paginação. 
Há uma página de login e cadastro que possibilitam a criação do usuário doador, após a criação é possível ver informações acerca do usuário e suas doações na página do perfil.

### USO DE I.A. : 
Foi utilizado I.A. para geração do front-end da aplicação, toda a estilização e parte da estrutura do HTML. 
Também foi utilizado para poder preencher o banco de projetos para fins de teste, deixarei o script python "seed.py" anexado junto ao repositório.
