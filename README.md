
# Projeto de site Crowdfunding.

A ideia é fazer um site que ofereça o modelo crowdfunding conectando os que precisam de doação a quem está disposto a doar algum recurso. 
Com ênfase em projetos voltados à juventude e às crianças. 

O modelo será uma plataforma que se poderá se registrar como doador ou projeto para que possam doar, com exposição dos projetos mais robustos e o progresso de sua meta.
